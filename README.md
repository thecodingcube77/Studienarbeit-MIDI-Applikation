# Info
Dieses Repository gehört zu einer Studienarbeit, welche an der DHBW Mannheim erarbeitet wurde.
# Abhängigkeiten
Das Programm hat eine Abhängigkeit von der Arduino-CLI. Diese wird zum Kompilieren und Flashen des Mikrocontroller Codes benötigt. Ohne die Arduino-CLI ist der Rest des Programmes aber auch funktionsfähig.
Die Arduino-CLI kann [hier](https://docs.arduino.cc/arduino-cli/installation/?_gl=1*f245k3*_up*MQ..*_ga*NDY5NzA0MjM1LjE3NzYxMjM0MjU.*_ga_NEXN8H46L5*czE3NzYxMjM0MjIkbzEkZzAkdDE3NzYxMjM0MjIkajYwJGwwJGgxMjk5OTc5NDY4) heruntergeladen werden.
# Start der Anwendung
Zum Starten der Anwendung muss der Befehl `mvn compile exec:java` ausgeführt werden. Dafür wird natürlich Maven benötigt.
